# 学习报告系统

老师端和开发者端完整的学习报告生成系统。

## 功能

- 老师端：查看和管理学生学习报告
- 开发者端：配置大纲、更新版本、管理内容

## 部署

本项目已配置为可部署到Railway平台。

### Railway部署步骤

1. Fork本仓库到您的GitHub
2. 登录 [Railway](https://railway.app)
3. 选择 "Deploy from GitHub repo"
4. 选择本仓库
5. 等待部署完成，获取访问链接

## 本地运行

```bash
npm install
npm start
```

访问 http://localhost:3000
