# 学习报告图片选择功能规范

## 为什么需要这个功能

当前系统中，学习报告生成页面的图片显示逻辑存在问题：用户上传了特定讲次（如第2讲、第3讲）的图片后，学习报告中会显示所有图片，而不是用户实际需要的图片。用户需要一个灵活的图片选择机制，能够：
1. 根据上传文件的命名自动匹配对应讲次的图片
2. 手动选择要在学习报告中显示的图片

## 需要做什么

### 1. 图片命名规范解析
系统需要能够解析图片文件名，提取关键信息：
- 学科：数学、语文
- 级别：L1、L2、L3、L4
- 讲次：第1讲、第2讲等
- 支持的格式：`数学-L1-第1讲.png`、`语文-L2-第3讲.jpg`、`语文-L4-第二讲.png`

### 2. 图片自动过滤功能
当用户选择学科和级别后：
- 系统自动扫描已上传的图片文件
- 根据文件名匹配当前选择的讲次范围
- 仅显示匹配的图片
- 例如：用户选择"数学L1"，显示数学L1的所有5讲图片

### 3. 图片手动选择功能
用户可以选择：
- **自动模式**：根据文件名自动匹配
- **手动模式**：用户勾选需要显示的具体图片
- 支持全选/取消全选
- 显示选中图片的数量

### 4. 学习报告生成集成
在生成学习报告时：
- 仅使用用户选择/匹配的图片
- 图片按讲次顺序排列显示
- 记录用户的选择偏好

## 影响范围

### 受影响的文件
- `teacher.js` - 需要添加图片解析和选择逻辑
- `index.html` - 需要添加图片选择UI组件
- `teacher.css` - 添加图片选择器样式

### 新增功能
- 图片文件名解析器
- 图片选择器组件
- 自动过滤逻辑
- 学习报告生成时的图片筛选逻辑

## 功能需求

### 需求1: 图片文件命名规范解析
系统**必须**能够正确解析以下格式的图片文件名：
- `数学-L1-第1讲.png`
- `语文-L2-第3讲.jpg`
- `语文-L4-第二讲.png`
- `数学-L3-第5讲.gif`

#### 场景：文件名解析成功
- **当** 用户上传名为 `数学-L1-第2讲.png` 的图片
- **那么** 系统应识别出：学科=数学，级别=L1，讲次=第2讲

#### 场景：文件名解析失败
- **当** 图片文件名不符合规范格式
- **那么** 系统应跳过该图片，不显示错误

### 需求2: 自动模式下的图片过滤
系统**应该**在自动模式下根据用户选择的学科和级别自动过滤图片。

#### 场景：自动过滤数学L1图片
- **当** 用户选择学科=数学，级别=L1
- **并且** 图片文件夹中存在 `数学-L1-第1讲.png`、`数学-L1-第2讲.png`、`数学-L2-第1讲.png`
- **那么** 系统仅显示匹配的文件：`数学-L1-第1讲.png`、`数学-L1-第2讲.png`

### 需求3: 手动模式下的图片选择
系统**应该**支持用户手动选择要显示的图片。

#### 场景：用户手动选择图片
- **当** 用户切换到手动模式
- **那么** 系统应显示所有可用图片，并提供勾选框
- **并且** 用户可以勾选任意图片组合
- **并且** 显示已选择图片的数量（如"已选择 3/5 张图片"）

### 需求4: 学习报告图片集成
系统**必须**在生成学习报告时使用用户选择/匹配的图片。

#### 场景：生成包含选定图片的学习报告
- **当** 用户完成了图片选择（自动或手动）
- **并且** 用户点击"生成学习报告"
- **那么** 学习报告中仅包含用户选择/匹配的图片
- **并且** 图片按讲次顺序排列

### 需求5: 选择偏好保存
系统**应该**记住用户的选择偏好，下次访问时自动应用。

#### 场景：保存用户选择偏好
- **当** 用户完成图片选择并关闭页面
- **那么** 系统应保存当前的选择偏好（学科、级别、选中的图片列表）
- **并且** 下次访问时自动恢复这些设置

## 技术实现要点

### 1. 文件名解析正则表达式
```javascript
// 匹配格式：学科-级别-讲次.扩展名
const pattern = /^(数学|语文)-L(\d)-第?(\d+)讲\.(png|jpg|jpeg|gif)$/i;
```

### 2. 图片过滤逻辑
```javascript
function filterImagesBySubjectLevel(images, subject, level) {
    return images.filter(img => {
        const parsed = parseImageName(img);
        return parsed && parsed.subject === subject && parsed.level === level;
    });
}
```

### 3. 本地存储键名
- `learningReportSubject` - 选择的学科
- `learningReportLevel` - 选择的级别
- `learningReportMode` - 选择模式（auto/manual）
- `learningReportSelectedImages` - 手动选择的图片列表
